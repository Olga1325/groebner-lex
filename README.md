# Экспериментальное исследование влияния порядка переменных на время вычисления базиса Грёбнера

## Описание проекта

Данный репозиторий посвящён экспериментальному исследованию **влияния порядка переменных** на производительность вычисления базиса Грёбнера полиномиальных идеалов с использованием библиотеки **GInvDist**.

Основная цель эксперимента — оценить, насколько сильно изменение порядка переменных (в частности, выбор ведущей переменной) влияет на время вычисления и стабильность результатов.

### Основная идея эксперимента

Для каждого теста мы фиксируем **100 прогонов** для каждой возможной ведущей переменной:
- Каждая из переменных поочерёдно ставится **на первую позицию** ровно 100 раз.
- Остальные позиции после первой заполняются случайной перестановкой оставшихся переменных.
- Таким образом, для системы с *n* переменными генерируется **n × 100** порядков.

Такой подход позволяет:
- Получить статистически значимое распределение времени для каждой ведущей переменной.
- Выявить «хорошие» и «плохие» ведущие переменные.
- Оценить чувствительность алгоритма к выбору порядка переменных.

## Структура эксперимента

- **Тестовые системы**: `geneig`, `comb3000` и другие.
- **Метод**: Инволютивный алгоритм библиотеки **GInvDist**.
- **Порядок мономов**: `TOPdeglex` (по умолчанию).
- **Количество прогонов на ведущую переменную**: **100**.
- **Общее количество порядков** для системы с *n* переменными: **n × 100**.

## Результаты эксперимента

Для каждого теста были получены:

- Полные логи выполнения (JSON-файлы для каждого порядка).
- Сводные таблицы результатов (`*.csv`).
- Статистика по времени, количеству критических пар и потреблению памяти.
- Графики зависимости времени вычисления от ведущей переменной (отдельный график для каждой позиции).

Особое внимание уделялось **позиции 1** (ведущая переменная), так как именно она оказывает наибольшее влияние на время вычисления в лексикографоподобных порядках.
